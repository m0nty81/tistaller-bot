📱 Keet Messenger
Установка и настройка для Android

Keet — это P2P мессенджер с сквозным шифрованием, который не использует серверы. Для стабильной работы важно правильно настроить работу приложения в фоне.

📥 ЧАСТЬ 1: УСТАНОВКА

1Скачайте приложение
  • Для устройств с Google Play:
    • Откройте Google Play
    • Введите в поиске "Keet" или "Keet - Private Encrypted Chat"
    • Нажмите "Установить"
  • Для устройств Huawei (без Google Play):
    • Откройте AppGallery (магазин приложений Huawei)
    • Введите в поиске "Keet"
    • Нажмите "Установить"
  • Альтернативный способ:
    • Посетите сайт keet.io/download
    • Скачайте APK-файл для Android
    • Разрешите установку из неизвестных источников при необходимости
Keet - мессенджер для конфиденциальности
2Запустите приложение
  • Откройте Keet после установки
  • Выберите "I'm new to Keet" (Я новый пользователь)
3Настройте профиль
  • Quick setup — фраза восстановления генерируется и автоматически сохраняется на устройстве
  • Manual setup — вы должны самостоятельно записать фразу восстановления
  • Запишите 24 слова фразы восстановления — это единственный способ восстановить доступ!
4Завершите настройку
  • Введите отображаемое имя (Display Name) — можно изменить в любой момент
  • При желании укажите имя пользователя (Username) — для поиска другими пользователями
📝 Правила для имени пользователя (Username):
  • Длина: от 4 до 31 символа (больше 3 и меньше 32)
  • Разрешены только строчные буквы латинского алфавита (a–z)
  • Разрешены цифры (0–9) — минимум одна обязательна
  • Можно использовать нижнее подчёркивание (_)
  • Пробелы, заглавные буквы и специальные символы запрещены
  • Должно быть уникальным в системе
  • Нельзя удалить — можно только изменить (максимум 5 попыток всего)
  • Старый username нельзя повторно использовать (ни вами, ни другими)
  • Примеры: user123, alex_smith7, johndoe2026

⚡ ЧАСТЬ 2: НАСТРОЙКА РАБОТЫ В ФОНЕ

⚠️ ВАЖНО: Android по умолчанию ограничивает работу приложений в фоне для экономии батареи. Без правильной настройки Keet может не получать сообщения!

🔧 Универсальная настройка (все Android)

1Откройте настройки приложения
  • Настройки → Приложения → Keet
2Настройте батарею
  • Нажмите "Батарея" или "Battery"
  • Выберите "Неограниченно" (Unrestricted) или "Без оптимизации"
  • Включите "Разрешить фоновую активность"
Настройки батареи:
Samsung: Настройки → Обслуживание устройства → Батарея → Приложения
Xiaomi: Настройки → Приложения → Управление приложениями → Keet → Батарея
Huawei: Настройки → Батарея → Запуск приложений
Tecno: Настройки → Battery Lab → Настройки экономии батареи
3Отключите оптимизацию батареи
  • Настройки → Приложения → Keet → Батарея
  • Нажмите «Оптимизация батареи»
  • Переключите на «Все приложения»
  • Найдите Keet и выберите «Не оптимизировать»
Отключение оптимизации:
Samsung: Настройки → Приложения → Keet → Батарея → Оптимизация батареи → Не оптимизировать
Xiaomi: Настройки → Приложения → Keet → Батарея → Экономия заряда в фоне → Без ограничений
Huawei: Настройки → Приложения → Keet → Батарея → Не оптимизировать
Tecno: Phone Master → Power Marathon → Отключить Power Boost

📱 Настройки для разных производителей

🔴 Samsung (One UI)

Android 11-14:

  1. Настройки → Обслуживание устройства → Батарея
  2. Нажмите меню (⋮) → Настройки
  3. Отключите:
    • ✓ «Переводить неиспользуемые приложения в спящий режим»
    • ✓ «Автооптимизация ежедневно»
    • ✓ «Адаптивная экономия заряда»
  4. Вернитесь → «Спящие приложения»
  5. Удалите Keet из списка спящих
  6. Добавьте Keet в «Приложения, которые не переводятся в спящий режим»

Ограничения в фоновом режиме:

  1. Настройки → Обслуживание устройства → Батарея → Ограничения в фоновом режиме
  2. Проверьте, что Keet нет в списках:
    • ❌ Приложения в режиме сна
    • ❌ Приложения с глубоким сном

Дополнительно (Android 14+):

  • Установите Good Guardians из Galaxy Store
  • Включите Memory Guardian → Режим «Быстрое переключение»
  • В Battery Guardian отключите экономию для Keet
Samsung Device Care (Обслуживание устройства):
Настройки → Обслуживание устройства → Батарея
Здесь доступны: Спящие приложения, Ограничения в фоновом режиме, Настройки батареи

🔵 Xiaomi (MIUI/HyperOS)

1. Автозапуск:

  1. Откройте приложение Безопасность (зелёная иконка с молнией)
  2. Разрешения → Автозапуск
  3. Включите Keet из списка

2. Контроль активности:

  1. Настройки → Приложения → Управление приложениями
  2. Найдите Keet → Контроль активности
  3. Выберите «Нет ограничений»

3. Батарея:

  1. Настройки → Батарея и производительность
  2. Экономия заряда в фоне → Выбор приложений
  3. Найдите Keet → «Без ограничений»

4. Закрепление в памяти:

  • Откройте Недавние приложения
  • Долгое нажатие на Keet → нажмите 🔒 Замок
  • ИЛИ свайп вниз на приложении до появления замка

Важно для MIUI:

  • ❌ Отключите «Расширенную оптимизацию» в настройках для разработчиков
  • ✅ Добавьте Keet в «Исключения» в приложении Очистка
Xiaomi MIUI/HyperOS:
Безопасность → Разрешения → Автозапуск
Настройки → Приложения → Управление приложениями → Keet → Контроль активности → Нет ограничений

🟢 Huawei (EMUI)

1. Запуск приложений:

  1. Настройки → Батарея → Запуск приложений
  2. Найдите Keet → Отключите «Управлять автоматически»
  3. Включите все три переключателя вручную:
    • ✓ Автозапуск
    • ✓ Косвенный запуск
    • ✓ Работа в фоне

2. Оптимизация батареи:

  1. Настройки → Приложения → Приложения
  2. Найдите Keet → Батарея
  3. Выберите «Не оптимизировать»

3. Закрепление в памяти:

  • Откройте Недавние приложения
  • Свайп вниз на карточке Keet до появления иконки 🔒

Важно для EMUI:

  • ✅ Включите «Расширенный запуск» для Keet
  • ❌ Отключите «Умную экономию заряда» в настройках батареи
Huawei EMUI:
Настройки → Батарея → Запуск приложений → Управлять вручную
Включите: Автозапуск, Косвенный запуск, Работа в фоне

⚫ Tecno (HiOS)

1. Phone Master:

  1. Откройте приложение Phone Master
  2. Инструменты → Управление автозапуском
  3. Включите Keet из списка

2. Power Marathon:

  1. Phone Master → Power Marathon
  2. Отключите «Power Boost» (Ускорение батареи)

3. App Booster:

  1. Phone Master → App Booster
  2. Добавьте Keet (максимум 4 приложения)

4. Battery Lab:

  1. Настройки → Battery Lab → Настройки экономии батареи
  2. Отключите «Экономию батареи для приложений»

5. Закрепление в памяти:

  • Откройте Недавние приложения
  • Нажмите на стрелку вниз на карточке Keet
  • Выберите первую опцию с иконкой замка 🔒
Tecno HiOS:
Phone Master → Инструменты → Управление автозапуском
Phone Master → Power Marathon → Отключить Power Boost
Настройки → Battery Lab → Настройки экономии батареи

🔒 Закрепление приложения в памяти

1Закрепите в недавних приложениях
  • Откройте недавние приложения (квадратная кнопка или свайп снизу с удержанием)
  • Найдите окно Keet
  • Действие зависит от бренда:
    • Xiaomi: Долгое нажатие → 🔒 ИЛИ свайп вниз
    • Samsung: Нажать на иконку → «Оставить открытым»
    • Huawei: Свайп вниз до появления 🔒
    • Tecno: Стрелка вниз → 🔒
2Трюк с выключением экрана
  • Откройте Keet
  • Не переходите на главный экран
  • Сразу выключите экран (кнопкой питания)
  • Это предотвратит закрытие приложения системой
✅ Проверка: После настройки перезагрузите телефон и проверьте:
  • Keet запускается автоматически
  • Сообщения приходят вовремя
  • Приложение не закрывается в фоне

👥 ЧАСТЬ 3: РАБОТА С КОНТАКТАМИ

Keet использует децентрализованную систему контактов.
Вы можете связаться с пользователем по username (имени пользователя) или через QR-код.
Все сообщения зашифрованы сквозным шифрованием и передаются напрямую между устройствами (P2P).

📝 Способ 1: Поиск по username

1Откройте поиск
  • Запустите Keet
  • В верхней части экрана нажмите на строку поиска с текстом «Search Keet»
  • Введите точное имя пользователя (например, john2026)
Введите username в строку поиска для поиска пользователя
2Отправьте запрос на добавление
  • Найдите нужного пользователя в результатах поиска
  • Нажмите на профиль пользователя
  • Нажмите кнопку «DM Request»
  • Введите приветственное сообщение (его увидит получатель до принятия запроса)
  • Нажмите «Send»
⚠️ Важно: Username должен быть введён точно, включая цифры и подчёркивания. Регистр не имеет значения (все username в нижнем регистре).
3Дождитесь подтверждения
  • Пользователь получит уведомление о запросе в верхней части списка чатов
  • После подтверждения вы сможете обмениваться сообщениями
  • Запрос отображается в списке чатов до тех пор, пока не будет принят или отклонён

📷 Способ 2: Сканирование QR-кода

1Откройте сканер QR-кодов
  • В строке поиска нажмите на иконку QR-сканера 🏁
  • Или долгое нажатие на иконку приложения на главном экране Android
2Отсканируйте QR-код
  • Предоставьте доступ к камере, если требуется
  • Наведите камеру на QR-код собеседника
  • Keet автоматически распознает код и откроет профиль пользователя
  • Нажмите «DM Request» для отправки запроса
Наведите камеру на QR-код для быстрого добавления контакта
💡 Как получить свой QR-код:
Откройте Profile (Профиль) → Нажмите «Profile QR Code» 🏁.
Можно показать код на экране или сохранить как изображение и отправить собеседнику.

📨 Управление запросами

1Просмотр входящих запросов
  • Запросы на добавление отображаются в верхней части списка чатов
  • Вы увидите только приветственное сообщение до принятия запроса
  • Нажмите «Accept» ✅ для принятия или «Ignore» ❌ для отклонения
2Отмена исходящего запроса
  • Откройте чат с пользователем, которому отправили запрос
  • Нажмите на имя пользователя в верхней части экрана
  • В меню профиля выберите «Leave DM»
⚠️ Обратите внимание:
  • Вы не можете добавить пользователя, который уже есть в контактах
  • Отправленный запрос можно отозвать до подтверждения
  • После удаления контакта повторное добавление возможно
  • Удаление DM-чата необратимо — вся история переписки и файлы будут удалены с вашего устройства
🔒 Конфиденциальность:
Если вы не указали username, другие пользователи не смогут найти вас через поиск.
В этом случае единственный способ связаться — через QR-код профиля или если вы сами отправите запрос.
